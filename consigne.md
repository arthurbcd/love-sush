Sujet du projet
L'objectif du projet est de continuer les travaux effectués jusqu'à maintenant pendant les TDs en créant une application autour du thème des sushis. Il est à réaliser en groupe de 3 à 4 personnes.
 
Le "cahier des charges" de votre application devra être explicite. Il est possible que toutes les fonctionnalités de celui-ci ne soient pas implémentées. Dans ce cas, il faudra me prévenir avant la remise du projet.
Bien évidemment cette application devra gérer de la persistance ( i.e continuer de fonctionner après arrêt/relance de l'application).
 
Elle ne devra pas nécessiter d'autre installation que Tomcat et l'application.
 
Elle doit également fournir une API pour permettre d'effectuer les mêmes opérations que l'interface Homme/Machine à minima coté utilisateur.
 
La première étape du projet consiste à m'envoyer votre groupe par mail ainsi qu'une description sommaire du projet.
 
Bonus : une autre application pourra être fournie pour gérer les différents éléments de références ( sushis, recette, moyen de livraison, etc ) selon les données nécessaires à votre application.
Les éléments suivants sont à fournir :
Cahier des charges de l'application
Application(s) avec les sources utilisable directement dans Eclipse.
Manuel utilisateur
Javadoc
Documentation technique
Critères de notation
complétude du projet
contrôle de cohérence
qualité de code
commentaire
évolutivité
conception
La date de remise des projets est fixée au Jeudi 21 Mai.
